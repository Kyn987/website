# Kyn's Bots Website + Admin Panel

## Pages
- `/status` : Public bot status page
- `/admin/login` : Admin login
- `/admin/dashboard` : Private admin dashboard

## Setup
1. Install dependencies:
   npm install

2. Create a `.env` file based on `.env.example`:
   - Set your ADMIN_USER
   - Set your ADMIN_PASS

3. Start the server:
   npm start

4. Deploy it to Render once ready!
